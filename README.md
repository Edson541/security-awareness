# Security Awareness Starter Kit

A short mini-project designed for a classroom submission: security awareness notes, a small password-checker script, and example content for phishing awareness.

## Contents
- `SECURITY_AWARENESS.md`   concise security awareness guide for users
- `phishing_examples.md` — real-world style examples of phishing red flags (educational)
- `scripts/password_checker.py` — small local script to evaluate password strength (educational only)
- `CONTRIBUTING.md` — how to improve this repo
